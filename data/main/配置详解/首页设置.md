# 首页设置

首页设置（`home`）用于控制站点入口页（如 `index.html`）的展示方式：使用外部 HTML 文件，或根据配置动态生成首屏与快速开始内容。

## 配置结构

```javascript
home: {
  use_file: false,           // 是否使用外部 HTML 作为首页内容
  file_path: "/home.html",   // 首页内容文件路径（use_file 为 true 时生效）
  hero: { ... },             // 首屏区域（use_file 为 false 时使用）
  features: [ ... ],         // 特性列表
  get_started: { ... }       // 快速开始指南
}
```

## 可用配置项

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `use_file` | 布尔值 | `false` | 为 `true` 时加载 `file_path` 指定的 HTML 作为首页主体内容 |
| `file_path` | 字符串 | `"/home.html"` | 首页 HTML 路径（相对站点根或绝对路径） |
| `hero` | 对象 | 见下文 | 首屏标题、描述、按钮等（仅当 `use_file` 为 false 时生效） |
| `features` | 数组 | 见下文 | 特性卡片列表 |
| `get_started` | 对象 | 见下文 | 快速开始区块的开关、标题、步骤与按钮 |

## 详细说明

### use_file 与 file_path

- **use_file: false**（默认）：首页由系统根据 `hero`、`features`、`get_started` 在 `index.html` 中动态渲染。
- **use_file: true**：不渲染上述区块，改为通过 `file_path` 加载外部 HTML（如 `/home.html`）作为首页主体内容，便于完全自定义首屏。

```javascript
home: {
  use_file: true,
  file_path: "/home.html"
}
```

### hero（首屏）

当 `use_file` 为 `false` 时生效，控制首屏大标题区。

| 字段 | 类型 | 说明 |
|------|------|------|
| `title` | 字符串 | 主标题 |
| `subtitle` | 字符串 | 副标题 |
| `description` | 字符串 | 简短描述 |
| `logo` | 字符串 | Logo 图片路径 |
| `button_text` | 字符串 | 主按钮文案（如「查看文档」） |
| `button_link` | 字符串 | 主按钮跳转链接（如 `"/main/"`） |

### features（特性列表）

数组，每项为一张特性卡片：

| 字段 | 类型 | 说明 |
|------|------|------|
| `icon` | 字符串 | FontAwesome 图标类名（如 `"fas fa-feather"`） |
| `title` | 字符串 | 标题 |
| `description` | 字符串 | 描述 |
| `color` | 字符串 | 主题色名（如 `"blue"`, `"orange"`, `"green"`） |

### get_started（快速开始）

| 字段 | 类型 | 说明 |
|------|------|------|
| `enable` | 布尔值 | 是否显示快速开始区块 |
| `title` | 字符串 | 区块标题 |
| `steps` | 数组 | 步骤列表，每项 `{ title, description }` |
| `button_text` | 字符串 | 底部按钮文案 |
| `button_link` | 字符串 | 底部按钮链接 |

## 完整示例

```javascript
home: {
  use_file: false,
  file_path: "/home.html",
  hero: {
    title: "EasyDocument",
    subtitle: "简易静态文档系统",
    description: "一个轻量级、免编译的纯静态前端文档系统。简单添加 Markdown 或 HTML 文件，即可生成美观、结构化的文档网站。",
    logo: "/assets/img/logo.svg",
    button_text: "查看文档",
    button_link: "/main/"
  },
  features: [
    { icon: "fas fa-feather", title: "轻量高效", description: "无需编译构建，纯前端实现，简单易用且响应迅速", color: "blue" },
    { icon: "fas fa-paint-brush", title: "美观现代", description: "精致排版，现代设计，丰富交互效果", color: "orange" },
    { icon: "fas fa-cog", title: "灵活可配置", description: "丰富的配置选项，自定义网站外观、布局和功能", color: "green" }
  ],
  get_started: {
    enable: true,
    title: "快速开始使用",
    steps: [
      { title: "添加文档文件", description: "将 .md 或 .html 文件放入 /data 目录" },
      { title: "组织目录结构", description: "创建分类目录，添加 README.md 作为分类介绍" },
      { title: "自定义配置", description: "修改 config.js 调整网站外观和功能" },
      { title: "运行 build.py 脚本", description: "生成 path.json 目录文件" },
      { title: "打开浏览器访问", description: "打开 index.html 即可使用" }
    ],
    button_text: "立即开始",
    button_link: "/main/"
  }
}
```
