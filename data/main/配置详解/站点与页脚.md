# 站点与页脚配置

本文档详细介绍 EasyDocument 系统中站点基本信息（`site`）和页脚（`footer`）相关的配置选项，这些选项控制网站的基本元数据和页脚显示内容。

## 站点基本信息

站点基本信息配置控制网站的核心元数据，影响浏览器标签显示、搜索引擎优化(SEO)等。

### 可用配置项

| 配置项 | 类型 | 默认值 | 说明 |
|-------|------|-------|------|
| `name` | 字符串 | `"EasyDocument"` | 网站名称 |
| `title` | 字符串 | `"简易静态文档系统"` | 网站标题，显示在浏览器标签页 |
| `description` | 字符串 | `"一个轻量级、免编译的纯静态前端文档系统"` | 网站描述，用于SEO |
| `keywords` | 字符串 | `"文档,静态网站,Markdown,Alpine.js"` | 网站关键词，用于SEO |
| `base_url` | 字符串 | `"/"` | 网站基础 URL，部署在子目录时需修改 |

### 详细说明

#### name

网站名称，用于显示在界面各处，如顶部导航栏、浏览器标签等。

```javascript
site: {
  name: "技术文档中心" // 自定义网站名称
}
```

这是网站的主要标识，应当简洁明了，易于记忆。

#### title

网站标题，主要用于显示在浏览器标签页和搜索引擎结果中。

```javascript
site: {
  title: "项目技术文档与API参考" // 更详细的标题
}
```

标题可以比网站名称更详细，包含更多关于网站内容和用途的信息。

#### description

网站描述，主要用于搜索引擎优化(SEO)，显示在搜索结果中。

```javascript
site: {
  description: "提供项目架构说明、API文档、使用指南和最佳实践的综合文档系统"
}
```

应当清晰说明网站的内容和价值，但不宜过长（推荐150-160字符以内）。

#### keywords

网站关键词，用于搜索引擎优化，帮助搜索引擎理解网站内容。

```javascript
site: {
  keywords: "技术文档,API参考,开发指南,代码示例,最佳实践"
}
```

关键词应当与网站内容紧密相关，使用逗号分隔。

#### base_url

网站基础URL，用于处理网站部署在子目录时的路径问题。

```javascript
site: {
  base_url: "/docs/" // 网站部署在/docs/子目录下
}
```

- 部署在根目录时使用 `"/"`
- 部署在子目录时使用如 `"/docs/"` 的路径

## 页脚设置

页脚设置控制网站底部显示的信息，包括版权、技术支持、链接、备案及列布局。

### 可用配置项

| 配置项 | 类型 | 说明 |
|-------|------|------|
| `copyright` | 字符串 | 版权信息文本 |
| `powered_by` | 对象 | 技术支持信息（是否显示、文案、链接） |
| `links` | 数组 | 页脚「资源」区链接列表 |
| `beian` | 对象 | 备案信息（是否启用、备案号、显示位置） |
| `columns` | 数组 | 页脚中部各列（导航、资源、技术栈等）的展示与开关 |

### 详细说明

#### copyright

页脚显示的版权信息文本。

```javascript
footer: {
  copyright: "© 2025 公司名称，保留所有权利"
}
```

#### powered_by

控制是否显示「使用以下技术栈构建」整块及链接。

```javascript
footer: {
  powered_by: {
    enable: true,
    text: "使用以下技术栈构建：",
    links: [
      { text: "TailwindCSS", url: "https://tailwindcss.com", external: true },
      { text: "Alpine.js", url: "https://alpinejs.dev", external: true }
    ]
  }
}
```

- `enable`: 是否显示该行
- `text`: 标题文案
- `links`: 技术栈链接，`external: true` 表示新标签打开

#### links

页脚「资源」区的链接列表，每项包含 `text`、`url`，可选 `external`。

```javascript
footer: {
  links: [
    { text: "GitHub", url: "https://github.com/yourname/yourrepo" },
    { text: "报告问题", url: "https://github.com/yourname/yourrepo/issues" }
  ]
}
```

#### beian

备案信息（如 ICP）。

```javascript
footer: {
  beian: {
    enable: true,
    icp: [
      { text: "京ICP备xxxxxxxx号", url: "https://beian.miit.gov.cn/" }
    ],
    position: "bottom"  // "top" 显示在版权附近；"bottom" 显示在页脚最底部
  }
}
```

- `icp`: 最多 1～3 条，每项为 `{ text, url? }`
- `position`: `"top"` 或 `"bottom"`

#### columns

控制页脚中部各列的展示与类型。

```javascript
footer: {
  columns: [
    { key: "nav", title: "导航", type: "nav", enable: true },
    { key: "links", title: "资源", type: "links", enable: true },
    { key: "stack", title: "技术栈", type: "stack", enable: true }
  ]
}
```

每列可单独通过 `enable: false` 隐藏。

## 常见用例

### 企业文档

适合企业内部或产品文档：

```javascript
site: {
  name: "产品名称",
  title: "产品技术文档中心",
  description: "提供产品架构、API参考和使用指南的官方文档",
  keywords: "产品名称,技术文档,API,用户指南",
  base_url: "/"
},
footer: {
  copyright: "© 2025 公司名称，保留所有权利",
  powered_by: { enable: false },
  links: [
    { text: "官方网站", url: "https://company.com" },
    { text: "支持中心", url: "https://support.company.com" }
  ]
}
```

### 开源项目

适合开源项目文档：

```javascript
site: {
  name: "项目名称",
  title: "开源项目文档",
  description: "一个功能强大的开源工具，简化开发流程",
  keywords: "开源,工具,开发者,文档",
  base_url: "/"
},
footer: {
  copyright: "© 2025 项目贡献者",
  powered_by: { enable: true, text: "使用以下技术栈构建：", links: [/* ... */] },
  links: [
    { text: "GitHub", url: "https://github.com/org/project" },
    { text: "报告问题", url: "https://github.com/org/project/issues" }
  ]
}
```

### 个人文档

适合个人知识库或博客：

```javascript
site: {
  name: "我的知识库",
  title: "个人技术笔记与教程",
  description: "分享Web开发、人工智能和系统架构的学习笔记",
  keywords: "编程,教程,笔记,学习资源",
  base_url: "/"
},
footer: {
  copyright: "© 2025 您的名字",
  powered_by: { enable: true },
  links: [
    { text: "个人网站", url: "https://yoursite.com" },
    { text: "Twitter", url: "https://twitter.com/youraccount", external: true }
  ]
}
```

## SEO优化建议

良好的站点配置可以显著提高搜索引擎可见性：

1. **精心撰写description**：清晰描述网站内容和价值
2. **选择合适的keywords**：使用与内容相关的关键词，避免过度堆砌
3. **利用title传递信息**：包含关键信息和品牌名称
4. **一致的site.name**：确保在所有相关资料中使用一致的名称

## 自定义页脚 HTML

顶栏/底栏也可通过自定义 HTML 文件渲染。在 `layout` 中设置：

```javascript
layout: {
  show_footer: true,
  footer_file: "/footer.html"
}
```

在项目根目录或对应路径创建 `footer.html`，编写所需 HTML 内容。

## 完整配置示例

```javascript
// 站点基本信息
site: {
  name: "EasyDocument",
  title: "简易静态文档系统",
  description: "一个轻量级、免编译的纯静态前端文档系统",
  keywords: "文档,静态网站,Markdown,Alpine.js",
  base_url: "/"
},

// 页脚设置
footer: {
  copyright: "© 2025 EasyDocument",
  powered_by: {
    enable: true,
    text: "使用以下技术栈构建：",
    links: [
      { text: "TailwindCSS", url: "https://tailwindcss.com", external: true },
      { text: "Alpine.js", url: "https://alpinejs.dev", external: true }
    ]
  },
  links: [
    { text: "GitHub", url: "https://github.com/LoosePrince/EasyDocument" },
    { text: "报告问题", url: "https://github.com/LoosePrince/EasyDocument/issues" }
  ],
  beian: { enable: false, icp: [], position: "bottom" },
  columns: [
    { key: "nav", title: "导航", type: "nav", enable: true },
    { key: "links", title: "资源", type: "links", enable: true },
    { key: "stack", title: "技术栈", type: "stack", enable: true }
  ]
}
```
